# 🛒 Retail Sales Data Analysis

## 📌 Project Overview
This project analyzes retail sales data to uncover insights about sales performance, profitability, and customer trends.

## 🎯 Objectives
- Analyze overall sales and profit
- Identify top-performing regions and categories
- Understand monthly sales trends
- Discover high-profit products

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib / Seaborn

## 📂 Dataset
Superstore Sales Dataset (Kaggle)

## 🔍 Key Analysis Performed
- Data cleaning and preprocessing
- Sales and profit aggregation
- Region-wise performance analysis
- Product-level insights
- Time-series analysis

## 📊 Key Insights
- The West region generates the highest profit
- Technology category performs best in terms of sales
- Certain products have high sales but low profit
- Seasonal trends observed in monthly sales
